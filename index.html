Main Projects

Collaborative Robot System Development (Stereo Depth Camera Based)

Independently developed a collaborative robot system based on ROS2 / Moveit 2, implementing real-time obstacle avoidance and human-robot collaboration algorithms using a stereo depth camera and robot arm. Utilized Gazebo for simulation environment setup, testing, and validation of the HRC system algorithms. Employed a stereo depth camera and human skeleton identification models to estimate 3D human skeletons. Calculated the positional relationship between human skeletons and the robot arm, controlling the robot arm via ROS2 Moveit 2 to enable real-time path and velocity replanning for safe human-robot collaboration.

ROSBot Autonomous Mobile Robot Development & Comparative Analysis

Developed ROSBot mobile robots based on both XITO framework and ROS2 platform, implementing high-level control, communication, and sensor integration. Conducted a systematic comparison of functionalities, performance, and scalability. Repaired and redesigned ROSBot mechanisms and control circuits to improve hardware stability and scalability. Developed dedicated Bash scripts for ROSBot to automate cruising demonstration tasks, enhancing efficiency and stability. Participated in evaluating the new automated control framework XITO, conducting technical tests and feasibility analysis, and providing decision-making recommendations.

Personal Projects

Self-Balancing Robot

Robot Competition Kit Full-Stack Development

Omniwheels

Designed and integrated robust omniwheels featuring **carbon fiber and 6061 aluminum construction** for enhanced durability and multi-directional movement. Multiple wheels were used to increase the overall rigidity and hardness of the robot's drive system.

Custom 4-in-1 ToF Sensor Board

Developed a specialized board for VL53L0X Time-of-Flight sensors, optimizing I2C address assignment and XSHUT pin control. This innovative solution enabled control of multiple sensors using only 2 pins via a logic gate, significantly reducing pin count requirements for the main controller.

Fire Extinguishing Robot

Developed an autonomous fire extinguishing robot designed to detect and suppress fires. Integrated sensors for flame detection and obstacle avoidance, with a custom control system to navigate and deploy extinguishing agents effectively.
